# 텔레그램 용어



### 사용방법

1. `Bot.batch` 더블클릭
2. 봇 생성
3. `Token`을 `resource/token.txt`에 저장
4. 공개 채널 생성
5. 봇을 채널에 연결



### 봇을 채널에 연결하기

1. 공개 채널 생성
2. 공개 채널에 봇 초대
3. /link 'channel_id' 입력하여 봇에게 채널 주소를 알려준다
4. (비공개 채널로 수정 가능)

### 조건식 받기

1. /broadcast on/off 채널에 방송유무 설정
2. /a 으로 첫 번째 조건식 결과 획득
3. /b 으로 두 번째 조건식 결과 획득
4. ..
